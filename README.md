# sweetcook-back

This is the backend part of my projet Sweetcook. Be sure to `npm start` this part to use the front end part.

## Getting started

```bash
$ npm install
$ npm start
```

## Don't forget to add .env and update it with

- PORT=
- DB_HOST=
- DB_USER=
- DB_PASSWORD=
- DB_NAME=
- SECRET_KEY=
